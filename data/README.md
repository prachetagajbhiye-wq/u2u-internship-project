# Data Directory

## Purpose

This folder contains all project-related datasets, study materials, and knowledge base resources.

## Contents

- PDF Documents
- DOCX Files
- PPT/PPTX Files
- Textbooks
- Notes
- Previous Year Question Papers (PYQs)
- Processed Chunks
- Embedding Data References

## Document Processing Workflow

Upload Document
→ Parse
→ Clean
→ Chunk
→ Generate Embeddings
→ Store in Vector Database

## Supported Formats

- PDF
- DOCX
- PPTX
- TXT
